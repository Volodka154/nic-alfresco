# nic-alfresco

Репозиторий содержит рабочее окружение Alfresco (Tomcat + кастомные модули),

используемое для разработки и отладки СЭД.

**ВАЖНО**: в репозитории используются большие бинарные файлы (`.war`),
которые хранятся через **Git Large File Storage (Git LFS)**.

Перед клонированием репозитория необходимо **установить Git LFS**.

```bash
git lfs version
